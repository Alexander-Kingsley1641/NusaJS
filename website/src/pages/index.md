# NusaJS

Framework JavaScript berbahasa Indonesia untuk web dan bot Discord.
Ringan, mudah dibaca, dan fokus pada alur kerja yang sederhana.

## Kenapa NusaJS?

- Bahasa dan istilah yang terasa lokal
- Runtime kecil untuk DOM dan bot
- Compiler minimal yang tetap bisa diperluas

## Mulai Cepat

```bash
npm create nusajs@latest web my-web
cd my-web
npm install
npm run dev
```
