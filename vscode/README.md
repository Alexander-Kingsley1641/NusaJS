# NusaJS VS Code Extension

⚠️ Pre-Alpha

Ekstensi ini menambahkan syntax highlighting untuk file `.jsi`.

## Fitur

- Highlight directive `@`
- Highlight keyword JavaScript utama
- Dukungan komentar dan string

## Instalasi Lokal

1. Buka folder `vscode/` di VS Code.
2. Jalankan perintah `Extensions: Install from VSIX...` jika sudah dibundle.
3. Atau gunakan `F5` untuk menjalankan Extension Development Host.
